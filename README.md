# Currency Converter

This project is part of a coding challenge. Currency converter app developed with the **MERN** stack (MongoDB, ExpressJS, React, Node )

## Installation

- Clone this repo
- Run npm install
- Start your MongoDB
- Npm start

## Responsiveness

You can see this app in all devices!
